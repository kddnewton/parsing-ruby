[What is ripper?]

[Requirements]

  * ruby 1.6 or 1.7 (I strongly recommend 1.7)
  * bison 1.28

[Compile and Test]

  $ ruby extconf.rb
  $ make
  $ ruby test.rb

[Installation]

  Copy ripper.{rb,so} to RUBYLIB.

[Usage]

  See samples.
  To know thrown events, see ripper.rb.
